# 命令行新建本地仓库并远程关联

```bash
git init
git add .
git commit -m "first commit"
git branch -M master
git remote add origin <repository-url>
git push -u origin master
```

# 命令行远程关联本地已有仓库

```bash
git remote add origin <repository-url>
git branch -M master
git push -u origin master
```
